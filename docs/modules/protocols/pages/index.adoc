== Overview

.High level overview of the protocols flow
[plantuml,format=svg]
....
| HTTP |
  :pair user;
| HTTPS |
  :app list;
  :start app;
| RTSP |
  :exchange next protocols info;
  :PLAY;
| Control/RTP |
  repeat
    :get user inputs;
    :Video/Audio stream data;
  repeat while (connected?) is (yes)
  -> no;

stop
....
